✦ How to Use This Tool

01

Author Override (top of sidebar) lets you describe your own setting, archetype, flaw, and tone in plain language. When filled in, it overrides all dropdowns. Leave it blank to use the dropdowns normally.

02

Set your parameters using the dropdowns — expanded with grouped options for Setting, Archetype, Fatal Flaw, and Tone. Add a project or client name if you like; it appears on the cover and footers.

03

Toggle sections on or off using the Include in Bible switches. Turn off Supporting Cast or Antagonist for a protagonist-only bible, or keep all on for a full ensemble package.

04

Click Build Character Bible. The AI generates your full document in seconds — cover page, protagonist sheet, character arc, supporting cast, and antagonist, all formatted and ready.

05

Save as PDF opens your browser print dialog. Set destination to Save as PDF and margins to None or Minimum for the cleanest output. Use Chrome or Edge for best results.

06

Copy Text puts a clean, formatted plain-text version on your clipboard — ready to paste into an email, Google Doc, or Canva template for further design work.
